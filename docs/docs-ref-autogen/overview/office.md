# <a name="office-javascript-api-reference"></a>Referência de API do JavaScript do Office

A API do JavaScript para Office permite criar aplicativos da web que interagem com os modelos de objeto nos aplicativos de host do Office. Use esta seção para saber mais sobre as classes, métodos e outros tipos disponíveis para a criação de suplementos do Office.

- [Excel](https://docs.microsoft.com/javascript/api/excel?view=office-js)
- [OneNote](https://docs.microsoft.com/javascript/api/onenote?view=office-js)
- [Outlook](https://docs.microsoft.com/javascript/api/outlook?view=office-js)
- [Visio](https://docs.microsoft.com/javascript/api/visio?view=office-js)
- [Word](https://docs.microsoft.com/javascript/api/word?view=office-js)
- [API compartilhado](https://docs.microsoft.com/javascript/api/office?view=office-js)
